<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser (3004) → API (8085) | Proxy Vite/Nginx encaminha /api; credenciais e tokens cruzam |
| Container api → container db | DATABASE_URL na rede interna fb_testesfc_net |
| .env → containers | Segredos (JWT_SECRET, ENCRYPTION_KEY, SMTP) injetados via ambiente |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Information Disclosure | Access token via XSS | mitigate | AuthContext mantém token em memória React (não localStorage); refresh token só em cookie httpOnly |
| T-02-02 | Spoofing/CSRF | /api/auth/refresh | mitigate | Cookie refresh SameSite=Strict; CORS whitelist em ALLOWED_ORIGINS limitada a localhost:3003/3004 |
| T-02-03 | Information Disclosure | Segredos em .env versionado | mitigate | Apenas `.env.example` versionado (placeholders); `.env` real não comitado; ENCRYPTION_KEY documentada (P4) |
| T-02-04 | Tampering | Imagem Docker do backend | mitigate | `go mod download` com go.sum verificado; build CGO_ENABLED=0 reproduzível; sem deps fora de escopo |
| T-02-05 | Elevation of Privilege | CORS no proxy | mitigate | SecurityMiddleware do backend valida Origin; Vite proxy apenas em dev |
</threat_model>
